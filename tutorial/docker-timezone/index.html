Docker 设置时区

简介

Docker 默认时区是协调世界时，又称世界标准时间，简称 UTC

中国大陆、香港、澳门、台湾、蒙古国、新加坡、马来西亚、菲律宾、澳洲西部的时间与 UTC 的时差均为 +8，也就是 UTC+8。

中国标准时间又称为 CST(China Standard Time)

查看时区

可以使用 date 命令查看时区信息

更改时区

根据应用方式的不同修改 Docker 时区可分为三种方式

docker-compose

Dockerfile

docker run
